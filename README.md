# Montgomery-County-and-Bias-Incidents-Analysis
## By Jacob Dawson, a Early College Student at Montgomery College Rockville

A statistical analysis using models and EDA based on the MCPD Bias incident dataset and local census data. Main goals include but not limmetted to finding association of races, district, or age with counts of bias incidents or crime of incident, predicting future years of counts of bias incidents, logistic testing of predicting whether a bias is reported or not. 

## Tools Used
 * Google Colab Notebooks (Python)
 * Data Montgomery (Data Repository Website)
 * API-based programmatic data ingestion (Extracting Data)
 * US Cencus ACS - 5 (Local Cencus Data for Montgomery County Maryland)
 * MCPD Bias Incidents (Main Dataset of Bias Incidents reported to and taken by Montgomery County Police)

## Project Structure
**ALL DATA IS EXTRACTED VIA API DATA INGESTION IN COLAB NOTEBOOKS**
**eda/**: Exploratory Data Analysis mostly including graphs and simple analysis in documents
**analysis/**: Preparing for models and hypothesis testing. Documentation on found results
**models/**: Includes Logistc Regrression, Poisson Regression, Linear Regression, Chi-Square testing, and more. Findings are in documentation. 
**reports/**: Summary of key findings and results.
